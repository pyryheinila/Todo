Fileselector and todoLists

Page strarts at fileselector (todo lists). Get there always by pressing the headers TDL button. There you can basicly add edit and enter different todolists. When you click a todolists name it will open the todos. Clicking a todo will show todos info and links if there is some.

Editor and add new

Edit todos by pressing the pencil button. Add new by pressing the + button. The editor has three different input:

  • Name - shouldn't be left empty
  • Importance - should be a number - smaller importance number comes before the larger one in the list.
  • Info - add notes (not required)

Add links by pressing ctrl + q and adding in the links name and url.

Completed todos

Press the checkmark button to complete a todo. The todo will then appear in the completedtodos-section. There you can see what you have completed and you can also return a todo back to the todo list.

Login and sign up

If you want to store your todolist data for use with many different devices, you can sign up and log in with your devices. You can log in in the right selectors log in button which changes to log out button when you are logged in.
